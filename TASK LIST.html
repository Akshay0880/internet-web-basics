Q&A

1. What does HTML stand for?

  • A) HyperText Markup Language
  • B) Hyper Transfer Markup Language
  • C) Hyperlinks and Text Markup Language
  • D) Hyper Transfer Model Language

A) HyperText Markup Language

2. What is the purpose of the <head> tag in HTML?

  • A) To display the main content
  • B) To contain metadata like title, character set, and link to stylesheets
  • C) To create hyperlinks
  • D) To define headings

B) To contain metadata like title, character set, and link to stylesheets

3. What is an HTML editor used for?

  • A) To create animations
  • B) To edit and write HTML code
  • C) To design databases
  • D) To browse the web

B) To edit and write HTML code

4. Which is the correct HTML element for inserting a line break?

  • A) <p>
  • B) <br>
  • C) <lb>
  • D) <break>

B) <br>

5. Which tag is used to create a hyperlink in HTML?

  • A) <a>
  • B) <link>
  • C) <href>
  • D) <hyper>

A) <a>

6. How do you create an unordered list in HTML?

  • A) <ul>
  • B) <ol>
  • C) <li>
  • D) <list>

A) <ul>

7. Which of the following is an HTML attribute?

  • A) <p>
  • B) href
  • C) <div>
  • D) <span>

B) href

8. What is the default alignment of the <h1> heading in HTML?

  • A) Left
  • B) Center
  • C) Right
  • D) Justified

A) Left

9. Which tag is used to define a paragraph in HTML?

  • A) <p>
  • B) <para>
  • C) <pg>
  • D) <par>

A) <p>

10. How do you add a comment in HTML?

  • A) <!-- comment -->
  • B) // comment
  • C) # comment
  • D) * comment *

A) <!-- comment -->

11. Which tag is used to define a table in HTML?

C) <table>

12. Which tag is used for making text italic in HTML?

D) Both A (<i>) and B (<em>)

13. What does the align attribute do in the <img> tag in HTML?

B) Aligns the image to the left or right

14. How do you specify an external CSS file for a webpage in HTML?

C) <link href="style.css" rel="stylesheet">

15. Which tag is used to create a definition term in a definition list in HTML?

A) <dt>
